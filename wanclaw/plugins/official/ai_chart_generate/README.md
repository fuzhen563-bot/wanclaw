# 图表自动生成

## 基本信息

| 属性 | 值 |
|------|-----|
| 插件ID | wanclaw.ai_chart_generate |
| 类型 | 官方内置插件 |
| 版本 | 2.0.0 |
| 分类 | AI增强 |
